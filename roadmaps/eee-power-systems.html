Capstone Projects

1. Newton-Raphson Load Flow Solver

Write a Python script to construct the Y-bus matrix and solve power flow equations using the Newton-Raphson algorithm for an IEEE 9-bus system.

Python / NumPy Y-bus Formulation Load Flow Analysis Jacobian Solver

2. Transmission Line Protection Simulation

MATLAB / Simulink Fault Analysis Relay Coordination Power System Tool

Placement & Interview Strategy

Written tests, concept evaluations, and practical design challenges overview.

Resume Keywords

  • Power Systems Analysis
  • Transmission Line Modeling
  • Load Flow (Newton-Raphson)
  • Symmetrical & Sequence Faults
  • Power System Protection & Relays
  • FACTS Devices & Smart Grids

Core Interview Focus

  • Deriving sequence networks for a star-delta transformer bank.
  • Explaining the physical mechanism of the Ferranti effect in unloaded lines.
  • Describing the Gauss-Seidel and Newton-Raphson convergence differences.
  • Applying the equal area criterion to find the critical clearing angle of a fault.
  • Contrasting impedance, reactance, and mho relays for transmission line zones.

Standard Preparation

  • Review textbooks and past exam question databases.
  • Practice timed numerical solving and multiple-choice questions.
  • Develop comprehensive subject-wise summary sheets.
  • Participate in mock tests to analyze error diagnostics.
